## Deskripsi Aplikasi
Mini-project ini adalah **Sistem Manajemen Organisasi** sederhana yang digunakan untuk memantau stok barang di sebuah perusahaan. Aplikasi ini dapat menambahkan, mengubah, menghapus, dan menampilkan data barang dengan antarmuka yang sederhana namun interaktif.

---

## Fitur Utama
- **Authentikasi: "" Login Register.
- **Manajemen Pengguna:** Tambah, edit, hapus, dan lihat daftar barang.  
- **Pencarian:** Cari barang berdasarkan nama atau kategori.  
- **Validasi Input:** Memastikan data yang dimasukkan sesuai format (misal stok > 0).  
- **User-Friendly Interface:** Tampilan bersih dan responsif, mudah digunakan.
  **Fitur Activity log digunakan:** untuk record aktivitas apa saja.

---

## Teknologi yang Digunakan
- **Backend:** Laravel 12 (mengimplementasikan prinsip SOLID)  
- **Frontend:** React + Inertia.js + Tailwind CSS  
- **Database:** MySQL  
- **Tools Pendukung:** Livewire untuk form interaktif, Lucide Icons untuk ikon, dan Inertia.js untuk pengalaman SPA

---

## Kemanan
- Authentication Login/Register
- LogActivity
- Validation
---
## Tujuan Mini-Project
- Menunjukkan kemampuan membangun aplikasi full-stack sederhana.  
- Menerapkan arsitektur modern menggunakan Laravel + React.  
- Mengikuti best practice: validasi data, struktur kode rapi, dan UI/UX mudah digunakan.
